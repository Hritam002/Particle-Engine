# Particle Engine

A lightweight JavaScript particle engine for rendering particle effects in the browser.

## Features
- Modular particle system
- Supernova explosion simulation
- 3D rendering support
- Custom particle behaviors

## Project Structure

engine/
- Particle.js
- SupernovaEngine.js

render/
- Renderer3D.js

index.html

## How to Run

1. Clone the repository
2. Open `index.html` in a browser

## Future Improvements
- WebGL optimization
- Physics-based particle interactions
- UI controls for parameters
